# ARES

**Autonomous Recognition and Engagement System**

ARES is an AI-powered embedded guidance system developed for free-fall munitions. It integrates a custom-trained YOLOv8 object detection model with a real-time targeting algorithm to direct payloads toward high-density enemy positions.

---

## 🔧 Components

- 🧠 Custom YOLOv8 object detection model
- 🎯 Density-based guidance algorithm
- ⚙️ Hardware integration: Raspberry Pi 5, Arduino Mega, ESP32
